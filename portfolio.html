Sampling of Work

This memory game is an example of a full-stack project. Created with React and hosted on GitHub pages.

The player is challenged to click on each photo only once. Once a selection is made, the photos are redistributed on the page in a random order. The game ends when the player selects the same photo twice.

Please use this link to give it a try: The Clicky Game

The superhero app uses the GIPHY API to make a dynamic web page.
By clicking on the button or typing in the name of a superhero, the page will be populated with 15 static gifs. When clicked, a static gif will become dynamic. Please give the app a try at: Superheros

Devour the Burger is an example of a full-stack application which uses MySQL, Node, Express, Handlebars and a homemade ORM.

A burger is created when the player enters the name of a burger.

Once created, burgers are added to the “Burgers Available” column. When the player clicks the devour button the burger is moved to the “Devoured” column. The burger is deleted from the screen once the player presses the “Yum” button.

This project is deployed on Heroku and can be accessed at: Devour the Burger
